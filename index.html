EVE Player Data Database

What is E. P. D. D.

The EVE Player Data Database is an online tool that allows EVE players to see important data for their in game characters all in one place. This both helps players avoid the unsatisfactory in game display of this data while beating out other similar services by showing all relevant data in one easy to read, easy to use, and easy to understand page.
